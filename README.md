# RU Slider Block

Gutenberg slider met één parent block (**RU Slider**) en child blocks (**RU Slide**).  
Features: Infinite loop, pijlen, dots, Type A (linker kolom, px/%), Type B (caption met optionele hoogte), popup op klik, per slider hoogte (desktop/mobiel), per slide accentkleur, verborgen scrollbar. Geen build-stap.

## Installatie
1. Plaats de map `ru-slider-block/` in `wp-content/plugins/`.
2. Activeer **RU Slider Block**.
3. In de editor: voeg **RU Slider** toe → voeg binnenin één of meer **RU Slide** blocks toe.
4. Stel instellingen in de rechter zijbalk:
   - **Slider**: hoogte desktop/mobiel, dots, pijlen, infinite loop
   - **Slide**: layout A/B, afbeelding, accentkleur, kolombreedte (A), caption & hoogte (B), titel/tekst
